Tafl
====

This is an Elixir implementation of the ancient game of Tafl.

Depedencies
===========

This project uses Elixir, Mix, Cowboy, Ranch and JSEX.

Building
========

 - mix deps.get
 - mix deps.compile

Running
=======

Run 'mix test' to run the unit tests. Use 'iex -S mix' to run the interactive mode.
